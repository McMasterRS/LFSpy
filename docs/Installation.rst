Installation
============

LFSpy is available on the `Python Package Index (PyPI) <https://pypi.org/project/LFSpy/>`__.

To install LFSpy along with its dependencies run the command::

    pip install lfspy

Dependencies
------------

LFS requires:

* Python 3
* NumPy>=1.14
* SciPy>=1.1
* Scikit-learn>=0.18.2
* pytest>=5.0.0
